# 我们的电视

电视直播软件，安装即可使用


# 下载

https://github.com/andandroidor/ourtv/releases/

xxxx_tv.apk 电视版

xxxx_android.apk 安卓手机版

与央视频不兼容，建议关闭各应用市场的自动更新。可能会被央视频覆盖安装。

# 安装

-   U盘安装
-   电视如果启用开发者调试模式，可以通过ADB命令进行安装：

```
   adb  install  -r  -f   xxx.apk
```
